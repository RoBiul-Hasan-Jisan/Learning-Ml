✅ NumPy: Creating Arrays

✅ Example 1: Creating an array using a Python list

import numpy as np

arr = np.array([1, 2, 3, 4, 5])

print(arr)
print(type(arr))
[1 2 3 4 5]
<class 'numpy.ndarray'>

✅ Example 2: Creating an array using a Python tuple

import numpy as np

arr = np.array((1, 2, 3, 4, 5))

print(arr)
[1 2 3 4 5]
